 Day 1  Wireshark Fundamentals

## Objective
Learn how to capture and analyze network packets using Wireshark.

## Environment
- Operating System: Kali Linux
- Tool: Wireshark

## Topics Learned
- What is a Network
- What is a Packet
- Packet Capture
- Wireshark Interface
- ICMP Protocol
- Echo Request
- Echo Reply
- Display Filters

## Commands Used

ping google.com

## Display Filter

icmp

## Practical Exercise

I generated ICMP traffic using the ping command and filtered the packets in Wireshark. I successfully identified Echo Requests and Echo Replies between my computer and Google's server.

## What I Learned

Today I learned how Wireshark captures network traffic and how filters make it easier to analyze specific protocols. I now understand the basics of ICMP communication and packet analysis.

## Skills Gained

- Packet Analysis
- ICMP Analysis
- Wireshark Navigation
- Network Fundamentals
